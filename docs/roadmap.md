# Roadmap

## Phase 1 — Windows-safe host parity
- Milestone 0: feasibility spike
- Milestone 1: safe device control MVP
- Milestone 2: official flashing support
- Milestone 3: capture path MVP
- Milestone 4: public hardening

## Phase 2 — fuller parity
- Sniffing robustness improvements
- Better Wireshark UX
- Broader command parity
- Performance and timing hardening

## Success definition
Phase 1 succeeds when a new Windows user can install the tool, detect a stock device, run safe commands, use the official flashing flow with guardrails, and produce at least one documented capture/export workflow.
